第七课 平面向量

1. 向量基础知识

  1. 定义

    • 向量：既有大小，又有方向的量

    • 数量：只有大小，没有方向的量

    • 零向量：长度为 000 的向量，表示为 0⃗\vec{0}0，方向任意

    • 单位向量：长度等于 111 个单位的向量，即 ∣a⃗∣=1\vert\vec{a}\vert=1∣a∣=1

    • 相等向量：长度相等且方向相同的向量

    • 相反向量：长度相等且方向相反的向量，即 a⃗+b⃗=0⃗\vec{a}+\vec{b}=\vec{0}a+b=0

    • 平行向量：方向相同或相反的非零向量，零向量与任何向量平行，没有传递性

  2. 向量的表示

    • 几何表示（有向线段）

    • 小写字母（a⃗\vec{a}a）

    • 坐标（a⃗=(x,y)\vec{a}=(x,y)a=(x,y)）

2. 向量的运算

  1. 向量加法运算

    • 三角形法则：首尾相连，由首至尾

    • 平行四边形法则：共起点

    • 向量三角不等式：∣∣a⃗∣−∣b⃗∣∣≤∣a⃗±b⃗∣≤∣a⃗∣+∣b⃗∣\vert\vert\vec{a}\vert-\vert\vec{b}\vert\vert\le\vert\vec{a}\pm\vec{b}\vert\le\vert\vec{a}\vert+\vert\vec{b}\vert∣∣a∣−∣b∣∣≤∣a±b∣≤∣a∣+∣b∣

  2. 向量减法运算

    • 三角形法则：共起点，连终点，方向指向被减向量

  3. 向量数乘运算

    • 实数 λ\lambdaλ 与向量 a⃗\vec{a}a 的积是一个向量，记作 λa⃗\lambda\vec{a}λa
    • ∣λa⃗∣=∣λ∣∣a⃗∣\vert\lambda\vec{a}\vert=\vert\lambda\vert\vert\vec{a}\vert∣λa∣=∣λ∣∣a∣
    • 当 λ>0\lambda>0λ>0 时，λa⃗\lambda\vec{a}λa 的方向与 a⃗\vec{a}a 的方向相同，当 λ<0\lambda<0λ<0 时，λa⃗\lambda\vec{a}λa 的方向与 a⃗\vec{a}a 的方向相反，当 λ=0\lambda=0λ=0 时，λa⃗=0⃗\lambda\vec{a}=\vec{0}λa=0

  4. 向量的数量积和投影

    • a⃗⋅b⃗=∣a⃗∣∣b⃗∣cos⁡θ\vec{a}\cdot\vec{b}=\vert\vec{a}\vert\vert\vec{b}\vert\cos\thetaa⋅b=∣a∣∣b∣cosθ，零向量与任意向量的数量积为 000

    • θ=0°⇔a⃗\theta=0\degree\Leftrightarrow\vec{a}θ=0°⇔a 与 b⃗\vec{b}b 同向 ⇔a⃗⋅b⃗=∣a⃗∣∣b⃗∣\Leftrightarrow\vec{a}\cdot\vec{b}=\vert\vec{a}\vert\vert\vec{b}\vert⇔a⋅b=∣a∣∣b∣

    • θ=180°⇔a⃗\theta=180\degree\Leftrightarrow\vec{a}θ=180°⇔a 与 b⃗\vec{b}b 反向 ⇔a⃗⋅b⃗=−∣a⃗∣∣b⃗∣\Leftrightarrow\vec{a}\cdot\vec{b}=-\vert\vec{a}\vert\vert\vec{b}\vert⇔a⋅b=−∣a∣∣b∣

    • θ\thetaθ 为锐角 ⇔a⃗⋅b⃗>0\Leftrightarrow \vec{a}\cdot\vec{b}>0⇔a⋅b>0 且 a⃗\vec{a}a 与 b⃗\vec{b}b 不共线

    • θ\thetaθ 为钝角 ⇔a⃗⋅b⃗<0\Leftrightarrow \vec{a}\cdot\vec{b}<0⇔a⋅b<0 且 a⃗\vec{a}a 与 b⃗\vec{b}b 不共线

    • 投影：a⃗\vec{a}a 在 b⃗\vec{b}b 方向上的投影为 ∣a⃗∣cos⁡θ\vert\vec{a}\vert\cos\theta∣a∣cosθ 或 a⃗⋅b⃗∣b⃗∣\dfrac{\vec{a}\cdot\vec{b}}{\vert\vec{b}\vert}∣b∣a⋅b​，投影是一个数值，可正可负可零

    • 数量积的几何意义：a⃗\vec{a}a 的长度与 b⃗\vec{b}b 在 a⃗\vec{a}a 方向上的投影的乘积

    • 投影向量：a⃗\vec{a}a 在 b⃗\vec{b}b 方向上的投影向量为∣a⃗∣cos⁡θ⋅b⃗∣b⃗∣\vert\vec{a}\vert\cos\theta\cdot\dfrac{\vec{b}}{\vert\vec{b}\vert}∣a∣cosθ⋅∣b∣b​ 或 (a⃗⋅b⃗)b⃗b⃗2\dfrac{(\vec{a}\cdot\vec{b})\vec{b}}{\vec{b}^2}b2(a⋅b)b​

    • 性质：∣a⃗⋅b⃗∣≤∣a⃗∣∣b⃗∣\vert\vec{a}\cdot\vec{b}\vert\le\vert\vec{a}\vert\vert\vec{b}\vert∣a⋅b∣≤∣a∣∣b∣

    • 运算律1：a⃗⋅b⃗=b⃗⋅a⃗\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{a}a⋅b=b⋅a

    • 运算律2：(λa⃗)⋅b⃗=λ(a⃗⋅b⃗)=a⃗⋅λb⃗(\lambda\vec{a})\cdot\vec{b}=\lambda(\vec{a}\cdot\vec{b})=\vec{a}\cdot\lambda\vec{b}(λa)⋅b=λ(a⋅b)=a⋅λb

    • 运算律3：(a⃗+b⃗)⋅c⃗=a⃗⋅c⃗+b⃗⋅c⃗(\vec{a}+\vec{b})\cdot\vec{c}=\vec{a}\cdot\vec{c}+\vec{b}\cdot\vec{c}(a+b)⋅c=a⋅c+b⋅c

    • 注意：没有分配律！

3. 平面向量和坐标

  1. 向量的坐标

    选定 {i⃗,j⃗}\{\vec{i},\vec{j}\}{i,j​} 为基底（通常是垂直的），令 a⃗=(1,0)\vec{a}=(1,0)a=(1,0)，b⃗=(0,1)\vec{b}=(0,1)b=(0,1)

    若 a⃗=x1i⃗+y1j⃗\vec{a}=x_1\vec{i}+y_1\vec{j}a=x1​i+y1​j​，则把有序数对 (x1,y1)(x_1,y_1)(x1​,y1​) 叫做 a⃗\vec{a}a 的坐标

  2. 向量的坐标运算

    令 a⃗=(x1,y1)\vec{a}=(x_1,y_1)a=(x1​,y1​)，b⃗=(x2,y2)\vec{b}=(x_2,y_2)b=(x2​,y2​)

    • a⃗+b⃗=(x1+x2,y1+y2)\vec{a}+\vec{b}=(x_1+x_2,y_1+y_2)a+b=(x1​+x2​,y1​+y2​)
    • a⃗−b⃗=(x1−x2,y1−y2)\vec{a}-\vec{b}=(x_1-x_2,y_1-y_2)a−b=(x1​−x2​,y1​−y2​)
    • λa⃗=(λx1,λx2)\lambda\vec{a}=(\lambda x_1,\lambda x_2)λa=(λx1​,λx2​)
    • a⃗⋅b⃗=x1x2+y1y2\vec{a}\cdot\vec{b}=x_1x_2+y_1y_2a⋅b=x1​x2​+y1​y2​
    • cos⁡<a⃗,b⃗>=a⃗⋅b⃗∣a⃗∣∣b⃗∣=x1x2+y1y2x12+y12x22+y22\cos <\vec{a},\vec{b}>=\dfrac{\vec{a}\cdot\vec{b}}{\vert\vec{a}\vert\vert\vec{b}\vert}=\dfrac{x_1x_2+y_1y_2}{\sqrt{x_1^2+y_1^2}\sqrt{x_2^2+y_2^2}}cos<a,b>=∣a∣∣b∣a⋅b​=x12​+y12​​x22​+y22​​x1​x2​+y1​y2​​

4. 平面向量有关定理

  1. 向量共线定理

    • 向量 a⃗(a⃗≠0⃗)\vec{a}(\vec{a}\neq\vec{0})a(a​=0)，当且仅当存在唯一一个实数 λ\lambdaλ，使得 b⃗=λa⃗\vec{b}=\lambda\vec{a}b=λa

    • 推论1：AAA、BBB、CCC 三点共线 ⇔AB⃗\Leftrightarrow \vec{AB}⇔AB、AC⃗\vec{AC}AC 共线

    • 推论2：向量 PA⃗\vec{PA}PA、PB⃗\vec{PB}PB、PC⃗\vec{PC}PC 三终点 AAA、BBB、CCC 共线 ⇔\Leftrightarrow⇔ 存在实数 α\alphaα、β\betaβ，使得 PA⃗=αPB⃗+αPC⃗\vec{PA}=\alpha\vec{PB}+\alpha\vec{PC}PA=αPB+αPC，且 α+β=1\alpha+\beta=1α+β=1

  2. 基底

    • 如果 e1⃗\vec{e_1}e1​​、e2⃗\vec{e_2}e2​​ 是同一平面内的两个不共线的向量，呢么对于这个平面内的任意向量 a⃗\vec{a}a，有且只有一对实数 λ1\lambda_1λ1​、λ2\lambda_2λ2​，使得 a⃗=λ1e1⃗+λ2e2⃗\vec{a}=\lambda_1\vec{e_1}+\lambda_2\vec{e_2}a=λ1​e1​​+λ2​e2​​，其中 e1⃗\vec{e_1}e1​​、e2⃗\vec{e_2}e2​​ 是一组基底

    • 注意：零向量不能作为基底

  3. 极化恒等式

    • a⃗⋅b⃗=14[(a⃗+b⃗)2−(a⃗−b⃗)2]\vec{a}\cdot\vec{b}=\dfrac{1}{4}[(\vec{a}+\vec{b})^2-(\vec{a}-\vec{b})^2]a⋅b=41​[(a+b)2−(a−b)2]
    • 几何意义：以 a⃗\vec{a}a 和 b⃗\vec{b}b 为邻边的平行四边形的两条对角线长度的平方差乘以 14\dfrac{1}{4}41​
    • 使用特点：两向量的和或差的模已知时

  4. 对角线向量公式（斯坦纳公式）

    AB⃗⋅CB⃗=(AD⃗2+CB⃗2)−(AB⃗2+CD⃗2)2\vec{AB}\cdot\vec{CB}=\dfrac{(\vec{AD}^2+\vec{CB}^2)-(\vec{AB}^2+\vec{CD}^2)}{2}AB⋅CB=2(AD2+CB2)−(AB2+CD2)​
