第二章 排列与组合

  1. 四个基本计数原理

    • 加法原理

      将集合 SSS 划分为两两不相交的集合 S1,S2,⋯ ,SmS_1,S_2,\cdots,S_mS1​,S2​,⋯,Sm​，则 SSS 的对象数目可以通过确定它的每一个部分的数量并如此相加而得到：

      ∣S∣=∣S1∣+∣S2∣+⋯+∣Sm∣\boxed{\vert S\vert=\vert S_1\vert+\vert S_2\vert+\cdots+\vert S_m\vert} ∣S∣=∣S1​∣+∣S2​∣+⋯+∣Sm​∣​

    • 乘法原理

      令 SSS 是对象的有序对 (a,b)(a,b)(a,b) 的集合，第一个对象 aaa 来自大小为 ppp 的一个集合，而对于 aaa 的每一个选择，对象 bbb 有 qqq 种选择，于是，SSS 的大小为 p×qp\times qp×q：

      ∣S∣=p×q\boxed{\vert S\vert=p\times q} ∣S∣=p×q​

    • 减法原理

      令 AAA 是一个集合，而 UUU 是包含 AAA 的更大的集合，设

      A‾=U∖A={x∈U:x∉A}\boxed{\overline{A}=U\setminus A=\{x\in U:x\notin A\} } A=U∖A={x∈U:x∈/​A}​

      是 AAA 在 UUU 中的补。那么 AAA 中的对象数目 ∣A∣\vert A\vert∣A∣ 由下列法则给出：

      ∣A∣=∣U∣−∣A‾∣\boxed{\vert A\vert=\vert U\vert-\vert\overline{A}\vert} ∣A∣=∣U∣−∣A∣​

    • 乘法原理

      令 SSS 是一个有限集合，把它划分成 kkk 个部分使得每一部分包含的对象数目相同，于是，此划分中的部分由下述公式给出：

      k=∣S∣在一个部分中的对象数目\boxed{k=\dfrac{\vert S\vert}{\text{在一个部分中的对象数目} } } k=在一个部分中的对象数目∣S∣​​

  2. 集合的排列

    rrr 是正整数，令一个 nnn 元素集合的 rrr 排列表示 nnn 个元素中的 rrr 个元素的有序放置,用 P(n,r)P(n,r)P(n,r) 表示 nnn 个元素集合的 rrr 排列的数目。

    • 定理 2.2.1

      对于正整数 nnn 和 rrr，t≤nt\le nt≤n，有：

      P(n,r)=n×(n−1)×⋯×(n−r+1)=n!(n−r)!\boxed{P(n,r)=n\times(n-1)\times\cdots\times(n-r+1)=\dfrac{n!}{(n-r)!} } P(n,r)=n×(n−1)×⋯×(n−r+1)=(n−r)!n!​​

    • 定理 2.2.2

      nnn 个元素的循环 rrr 排列的数目是

      P(n,r)r=n!r⋅(n−r)!\boxed{\dfrac{P(n,r)}{r}=\dfrac{n!}{r\cdot (n-r)!} } rP(n,r)​=r⋅(n−r)!n!​​

  3. 集合的组合（子集）

    rrr 是非负整数，令一个 nnn 元素集合 SSS 的 rrr 子集表示由 SSS 的 nnn 个对象中的 rrr 个对象组成的子集。用 (nr)n\choose r(rn​) 表示 nnn 元素集合的 rrr 子集的数目。

    对每一个非负整数 nnn，下述事实成立：

    (n0)=1, (n1)=n, (nn)=1, (00)=1\boxed{ {n\choose 0}=1,\ {n\choose 1}=n,\ {n\choose n}=1,\ {0\choose 0}=1} (0n​)=1, (1n​)=n, (nn​)=1, (00​)=1​

    • 定理 2.3.1

      对于 0≤r≤n0\le r\le n0≤r≤n，有

      P(n,r)=r!(nr)\boxed{P(n,r)=r!{n\choose r} } P(n,r)=r!(rn​)​

      因此

      (nr)=n!r!(n−r)!\boxed{ {n\choose r}=\dfrac{n!}{r!(n-r)!} } (rn​)=r!(n−r)!n!​​

    • 定理 2.3.2

      对于 0≤r≤n0\le r\le n0≤r≤n，有

      (nr)=(nn−r)\boxed{ {n\choose r}={n\choose n-r} } (rn​)=(n−rn​)​

    • 定理 2.3.3（帕斯卡公式）

      对于所有满足 1≤k≤n−11\le k\le n-11≤k≤n−1 的整数 nnn 和 kkk，有

      (nk)=(n−1k)+(n−1k−1)\boxed{ {n\choose k}={n-1\choose k}+{n-1\choose k-1} } (kn​)=(kn−1​)+(k−1n−1​)​

    • 定理 2.3.4

      对于 n≥0n\ge0n≥0，有

      (n0)+(n1)+⋯+(nn)=2n\boxed{ {n\choose 0}+{n\choose 1}+\cdots+{n\choose n}=2^n} (0n​)+(1n​)+⋯+(nn​)=2n​

      且这个共同值等于 nnn 元素集合的子集数量。
