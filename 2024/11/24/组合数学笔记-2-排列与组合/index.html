第二章 排列与组合

  1. 四个基本计数原理

    • 加法原理

      将集合 SSS 划分为两两不相交的集合 S1,S2,⋯ ,SmS_1,S_2,\cdots,S_mS1​,S2​,⋯,Sm​，则 SSS 的对象数目可以通过确定它的每一个部分的数量并如此相加而得到：

      ∣S∣=∣S1∣+∣S2∣+⋯+∣Sm∣\vert S\vert=\vert S_1\vert+\vert S_2\vert+\cdots+\vert S_m\vert ∣S∣=∣S1​∣+∣S2​∣+⋯+∣Sm​∣

    • 乘法原理

      令 SSS 是对象的有序对 (a,b)(a,b)(a,b) 的集合，第一个对象 aaa 来自大小为 ppp 的一个集合，而对于 aaa 的每一个选择，对象 bbb 有 qqq 种选择，于是，SSS 的大小为 p×qp\times qp×q：

      ∣S∣=p×q\vert S\vert=p\times q ∣S∣=p×q

    • 减法原理

      令 AAA 是一个集合，而 UUU 是包含 AAA 的更大的集合，设

      A‾=U∖A={x∈U:x∉A}\overline{A}=U\setminus A=\{x\in U:x\notin A\} A=U∖A={x∈U:x∈/​A}

      是 AAA 在 UUU 中的补。那么 AAA 中的对象数目 ∣A∣\vert A\vert∣A∣ 由下列法则给出：

      ∣A∣=∣U∣−∣A‾∣\vert A\vert=\vert U\vert-\vert\overline{A}\vert ∣A∣=∣U∣−∣A∣

    • 乘法原理

      令 SSS 是一个有限集合，把它划分成 kkk 个部分使得每一部分包含的对象数目相同，于是，此划分中的部分由下述公式给出：

      k=∣S∣在一个部分中的对象数目k=\dfrac{\vert S\vert}{\text{在一个部分中的对象数目} } k=在一个部分中的对象数目∣S∣​

  2. 集合的排列

    rrr 是正整数，令一个 nnn 元素集合的 rrr 排列表示 nnn 个元素中的 rrr 个元素的有序放置,用 P(n,r)P(n,r)P(n,r) 表示 nnn 个元素集合的 rrr 排列的数目。

    • 定理 2.2.1

      对于正整数 nnn 和 rrr，t≤nt\le nt≤n，有：

      P(n,r)=n×(n−1)×⋯×(n−r+1)=n!(n−r)!P(n,r)=n\times(n-1)\times\cdots\times(n-r+1)=\dfrac{n!}{(n-r)!} P(n,r)=n×(n−1)×⋯×(n−r+1)=(n−r)!n!​

    • 定理 2.2.2

      nnn 个元素的循环 rrr 排列的数目是

      P(n,r)r=n!r⋅(n−r)!\dfrac{P(n,r)}{r}=\dfrac{n!}{r\cdot (n-r)!} rP(n,r)​=r⋅(n−r)!n!​

  3. 集合的组合（子集）

    rrr 是非负整数，令一个 nnn 元素集合 SSS 的 rrr 子集表示由 SSS 的 nnn 个对象中的 rrr 个对象组成的子集。用 (nr)n\choose r(rn​) 表示 nnn 元素集合的 rrr 子集的数目。

    对每一个非负整数 nnn，下述事实成立：

    (n0)=1, (n1)=n, (nn)=1, (00)=1{n\choose 0}=1,\ {n\choose 1}=n,\ {n\choose n}=1,\ {0\choose 0}=1 (0n​)=1, (1n​)=n, (nn​)=1, (00​)=1

    • 定理 2.3.1

      对于 0≤r≤n0\le r\le n0≤r≤n，有

      P(n,r)=r!(nr)P(n,r)=r!{n\choose r} P(n,r)=r!(rn​)

      因此

      (nr)=n!r!(n−r)!{n\choose r}=\dfrac{n!}{r!(n-r)!} (rn​)=r!(n−r)!n!​

    • 定理 2.3.2

      对于 0≤r≤n0\le r\le n0≤r≤n，有

      (nr)=(nn−r){n\choose r}={n\choose n-r} (rn​)=(n−rn​)

    • 定理 2.3.3（帕斯卡公式）

      对于所有满足 1≤k≤n−11\le k\le n-11≤k≤n−1 的整数 nnn 和 kkk，有

      (nk)=(n−1k)+(n−1k−1){n\choose k}={n-1\choose k}+{n-1\choose k-1} (kn​)=(kn−1​)+(k−1n−1​)

    • 定理 2.3.4

      对于 n≥0n\ge0n≥0，有

      (n0)+(n1)+⋯+(nn)=2n{n\choose 0}+{n\choose 1}+\cdots+{n\choose n}=2^n (0n​)+(1n​)+⋯+(nn​)=2n

      且这个共同值等于 nnn 元素集合的子集数量。

  4. 多重集合的排列

    多重集合：元素可以出现多次，如一个有 333 个 aaa，222 个 bbb，无穷多个 ccc 的多重集合 MMM 记为 M={3⋅a,2⋅b,∞⋅c}M=\{3\cdot a,2\cdot b,\infty\cdot c\}M={3⋅a,2⋅b,∞⋅c}

    • 定理 2.4.1

      设 SSS 是有 kkk 种不同类型元素的多重集合，每一个元素有无限重复数，SSS 的 rrr 排列数是 krk^rkr

    • 定理 2.4.2

      设 SSS 是有 kkk 种不同类型元素的多重集合，每一种类型的有限重复数是 n1,n2,⋯ ,nkn_1,n_2,\cdots,n_kn1​,n2​,⋯,nk​。设 SSS 的大小为 n=n1+n2+⋯+nkn=n_1+n_2+\cdots+n_kn=n1​+n2​+⋯+nk​。则 SSS 的排列数目等于

      n!n1!n2!⋯nk!\dfrac{n!}{n_1!n_2!\cdots n_k!} n1​!n2​!⋯nk​!n!​

    • 定理 2.4.3

      设 nnn 是正整数，n1,n2,⋯ ,nkn_1,n_2,\cdots,n_kn1​,n2​,⋯,nk​ 是正整数且 n=n1+n2+⋯+nkn=n_1+n_2+\cdots+n_kn=n1​+n2​+⋯+nk​。把 nnn 对象集合划分为 kkk 个有标签的盒子，第 iii 个盒子有 nin_ini​ 个对象，这时的划分方法数等于

      n!n1!n2!⋯nk!\dfrac{n!}{n_1!n_2!\cdots n_k!} n1​!n2​!⋯nk​!n!​

      若盒子没有标签，且 n1=n2=⋯=nkn_1=n_2=\cdots=n_kn1​=n2​=⋯=nk​，则划分数等于

      n!k!n1!n2!⋯nk!\dfrac{n!}{k!n_1!n_2!\cdots n_k!} k!n1​!n2​!⋯nk​!n!​

    通常将多重集合的子集成为组合而不是多重子集

      (r+k−1r)=(r+k−1k−1){r+k-1\choose r}={r+k-1\choose k-1} (rr+k−1​)=(k−1r+k−1​)

  6. 有限概率

    有一个实验 ε\varepsilonε，在进行这个实验时，它产生的结果是某有限集合中的一个，假设每一个结果是等可能的，则称这个实验是随机的，所有可能的结果成为这个实验的样本空间，并把它记作 SSS，SSS 是一个有限集合，如下面这个集合

    S=S1,s2,⋯ ,snS={S_1,s_2,\cdots,s_n} S=S1​,s2​,⋯,sn​

    当我们进行实验 ε\varepsilonε 时，每个 sis_isi​ 都有 1/n1/n1/n 的出现机会，所以说结果的概率是 1/n1/n1/n，写作

    Prob(si)=1/nProb(s_i)=1/n Prob(si​)=1/n

    一个事件就是样本空间 SSS 的一个子集 EEE

    在样本空间为 SSS 的实验中，事件 EEE 的概率定义为 SSS 中属于 EEE 的结果的比率，因此

    Prob(E)=∣E∣∣S∣Prob(E)=\dfrac{\vert E\vert}{\vert S\vert} Prob(E)=∣S∣∣E∣​

    于是很多概率问题可以用排列组合解决
