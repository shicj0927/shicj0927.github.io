第三课 集合

[返回目录][洛谷专栏链接]

1. 集合的概念

  1. 概念

    指定的某些对象的全体（研究对象是元素）

  2. 特性

    确定性，互异性，无序性

  3. 元素与集合的关系

    属于（∈\in∈）

    不属于（∉\notin∈/​）

  4. 集合的分类

    有限集，无限集，空集（）

  5. 常见数集

    R\RR 实数集

    Q\mathbb QQ 有理数集

    Z\ZZ 整数集

    N\NN 自然数集

    C\ComplexC 复数集

    N∗=N+\N^*=\N_+N∗=N+​ 正整数集

    R+\R^+R+ 正实数集

  6. 集合的表示方法

    • 列举法

      A={1,2,3}A=\{1,2,3\}A={1,2,3}

2. 集合间的关系

  1. 子集

    若对于 ∀x∈A\forall x\in A∀x∈A，都有 x∈Bx\in Bx∈B ，则称集合 AAA 是集合 BBB 的子集，记为 A⊆BA\subseteq BA⊆B

  2. 真子集

    若 A⊆BA \subseteq BA⊆B，但 ∃ x∈B\exists\ x\in B∃ x∈B，且 x∉Ax\notin Ax∈/​A，则称集合 AAA 是集合 BBB 的真子集，记为 A⫋BA\subsetneqq BA⫋B

  3. 空集

    不含任意元素的集合是空集，记为 ∅\varnothing∅

    空集是任何集合的子集，是任何非空集合的真子集

    ∅∈{∅}\varnothing\in\{\varnothing\}∅∈{∅} ：∅\varnothing∅ 为元素

    ∅⊆{∅}\varnothing\subseteq\{\varnothing\}∅⊆{∅}：∅\varnothing∅ 为集合

    ∅⫋{∅}\varnothing\subsetneqq\{\varnothing\}∅⫋{∅}：∅\varnothing∅ 为集合，空集是任何非空集合的真子集

  4. 子集个数问题

    若一个集合有 nnn 个元素，则子集有 2n2^n2n 个，真子集有 2n−12^n-12n−1 个，非空子集有 2n−12^n-12n−1 个，非空真子集有 2n−22^n-22n−2 个

3. 集合的运算

  1. 交集

    A∩B={x∣x∈A,and x∈B}A\cap B=\{x|x\in A,\text{and}\ x\in B\}A∩B={x∣x∈A,and x∈B}

  2. 并集

    A∪B={a∣x∈A,or x∉A}A\cup B=\{a|x\in A,\text{or}\ x\notin A\}A∪B={a∣x∈A,or x∈/​A}

  3. 补集

    ∁RA={x∣x∈R,and x∉A}\complement_\R A=\{x|x\in\R,\text{and}\ x\notin A\}∁R​A={x∣x∈R,and x∈/​A}

    性质（摩根公式）：

    ∁R(A∩B)=(∁RA)∪(∁RB)\complement_\R(A\cap B)=(\complement_\R A)\cup(\complement_\R B)∁R​(A∩B)=(∁R​A)∪(∁R​B)

    ∁R(A∪B)=(∁RA)∩(∁RB)\complement_\R(A\cup B)=(\complement_\R A)\cap(\complement_\R B)∁R​(A∪B)=(∁R​A)∩(∁R​B)

4. 充分条件与必要条件

  1. 定义

    “若 ppp，则 qqq”为真命题，记 p⇒qp\Rightarrow qp⇒q，则称 ppp 是 qqq 的充分条件，qqq 是 ppp 的必要条件

    “若 ppp，则 qqq”为假命题，记 p⇏qp\nRightarrow qp⇏q，则称 ppp 不是 qqq 的充分条件，qqq 不是 ppp 的必要条件

    “若 ppp，则 qqq”与它的逆命题“若 QQQ，则 ppp”均为真命题，记 p⇔qp\Leftrightarrow qp⇔q，则称 ppp 是 qqq 的充分必要条件，qqq 是 ppp 的充分必要条件（简称充要条件）

5. 全称量词与存在量词

  1. 定义

    全称量词（∀\forall∀）：对每个元素都成立

    存在量词（∃\exists∃）：对至少等于一个元素成立

  2. 否定

    全称量词命题的否定：∀x∈M,p(x)\forall x\in M,p(x)∀x∈M,p(x) 的否定是 ∃x∈M,¬p(x)\exists x\in M,\lnot p(x)∃x∈M,¬p(x)

    存在量词命题的否定：∃x∈M,p(x)\exists x\in M,p(x)∃x∈M,p(x) 的否定是 ∀x∈M,¬p(x)\forall x\in M,\lnot p(x)∀x∈M,¬p(x)
